# 好好吃饭 · 私房菜单

给家庭主厨使用的轻量点菜网页。朋友可以按菜系浏览菜单、选择菜品和份数、填写用餐日期与忌口；主厨工作台会接收订单，并把未完成订单自动拆解、合并成食材与调料采购清单。

## 功能

- 朋友点菜：分类浏览、购物篮、用餐信息与备注
- 主厨接单：查看订单、确认接单、标记完成
- 采购清单：按全部未完成订单自动汇总用量
- 数据持久化：订单保存在 Cloudflare D1
- 响应式界面：适配手机和桌面网页

## 本地运行

需要 Node.js 22.13 或以上版本。

```bash
npm install
npm run dev
```

常用命令：

- `npm run build`：构建生产版本
- `npm test`：验证菜单、订单接口与数据库迁移
- `npm run db:generate`：数据库结构变化后生成迁移

## 数据结构

菜单与每道菜的食材配方位于 `app/menu.ts`，订单结构位于 `db/schema.ts`。增删菜品时，只需同步维护菜单中的配方，采购清单会自动更新。
