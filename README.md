# 好好吃饭 · 私房菜单

给家庭主厨使用的轻量点菜网页。朋友可以按菜系浏览菜单、选择菜品和份数、填写用餐日期与忌口；主厨工作台会接收订单，并把未完成订单自动拆解、合并成食材与调料采购清单。

## 功能

- 朋友点菜：分类浏览、购物篮、用餐信息与备注
- 主厨接单：查看订单、确认接单、标记完成
- 采购清单：按全部未完成订单自动汇总用量
- 菜单管理：添加自定义菜式、上下架或删除
- 菜品图片：支持本地上传，或直接填写网络图片地址
- 数据持久化：订单保存在 Cloudflare D1
- 图片存储：上传的菜品照片保存在 Cloudflare R2
- 响应式界面：适配手机和桌面网页

## 本地运行

需要 Node.js 22.13 或以上版本。

```bash
npm install
npm run dev
```

常用命令：

- `npm run build`：构建生产版本
- `npm test`：验证菜单、订单接口与数据库迁移
- `npm run db:generate`：数据库结构变化后生成迁移

## 数据结构

固定菜单与配方位于 `app/menu.ts`；主厨在网页里添加的新菜保存在数据库中，上传图片存放在对象存储中。每道自定义菜的配方会直接参与采购清单汇总。
