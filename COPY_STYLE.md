# 阿德小厨房 · slogan 风格指南

> 这是通义千问生成点菜端 slogan 的长期参考，也是当前生成接口使用的风格基准。它记录阿德本人的语言习惯，后续新增参考语料时继续在这里维护。

## 1. 核心气质

阿德式 slogan 不是“这道菜很好吃”的广告语，而是主厨或菜品本人顺口说出的一句吐槽。它应该有具体菜品知识作底，再拐一个轻巧、出人意料的弯；有点冷、有点欠，但仍像朋友之间说话，不刻薄、不油腻。

最重要的判断标准：遮住菜名以后，熟悉这道菜的人仍能从食材、做法、地域或典故猜出它是谁。如果换到大多数菜上也成立，就太普通，必须重写。

## 2. 常用创作手法

- 借用熟悉的文学、歌词或流行句式，再用菜品事实完成反转。
- 从菜名的双关、网络语义或字面歧义里找梗。
- 把本来不常见的两种食材写成意外相遇，带一点拟人。
- 用主厨第一人称自嘲、解释或“先认了”的口吻。
- 让菜品本人说话，调侃它的固定出场场景或大众印象。
- 使用当下流行文化梗，但必须自然贴合菜品；没有合适的梗时宁可写生活化冷幽默。

每条只使用一个主要笑点，并把包袱尽量放在后半句。不要解释笑点。

## 3. 文字规则

- 通常 10–24 个中文字符，以自然、完整和有节奏为先，不再硬性限制为 7–14 字。
- 可以使用逗号、问号和感叹号，也可以在笑点需要时重复菜名。
- 可以用“我”“你”“它”等口语视角，像阿德当面端菜时随口说的。
- 不要求直接赞美味道，但笑点必须源自这道菜的真实食材、做法、来历或食用场景。
- 避免广告腔、文艺腔和通用 AI 套话，例如“一口上头”“好吃到停不下来”“今天就想吃这一口”。
- 避免为了押韵硬造双关、为了追热点强塞网络梗，也不要照抄参考句式只替换菜名。
- 可以调侃自己、食材和流行文化，不攻击具体客人或群体，不使用恶意、歧视或令人不适的梗。

## 4. 千问生成流程

生成时应在内部完成以下步骤，只输出最终选中的一句：

1. 从菜名、配方和步骤中找出 2–4 个这道菜独有的事实钩子。
2. 选择最自然的一种手法：反转、双关、拟人、主厨自嘲、场景错位或流行文化调侃。
3. 内部写出至少 3 个方向不同的候选，不沿用同一个句式换词。
4. 检查事实是否准确、笑点是否一眼能懂、是否只有这道菜适用。
5. 删除最像广告、最用力或最通用的候选，只返回最像阿德本人会说的那一句。

## 5. 阿德本人参考语料

- 大虾烧白菜：“白天不懂夜的黑，白菜不懂虾的鲜”
- 油焖大虾：“没有虾头，是因为我不做虾头男”
- 清炖牛肋条：“苹果和牛肉这辈子没想过能混在一块”
- 看心情红烧肉：“红烧肉我反正每次烧的都不太一样，看心情”
- 豉汁排骨：“其实我不止出现在早茶”
- 茄汁鲅鱼：“我总不可能把番茄包到鲅鱼饺子里吧”
- 回锅肉：“又是回锅肉？内娱真完了！”

这些例子用于学习节奏、视角和幽默机制，不能直接复制结构或为不相干的菜硬套同一个梗。
